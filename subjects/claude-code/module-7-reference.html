Module 7: Quick Reference

Skill anatomy, frontmatter fields, invocation control, and where things live.

Skill Directory Structure

FileRequiredPurpose
SKILL.mdYesFrontmatter + instructions. The skill entrypoint.
reference.mdNoDetailed lookup material loaded on demand.
template.mdNoOutput template for Claude to fill in.
scripts/NoScripts Claude can execute via bash. Only output enters context.

Frontmatter Fields

FieldDescription
nameBecomes the /slash-command. Lowercase letters, numbers, hyphens only. Max 64 chars.
descriptionAlways loaded into context. Claude uses this to decide when to invoke the skill automatically. Front-load the key use case. Truncated at 1,536 chars.
when_to_useAdditional trigger context. Appended to description, counts toward the 1,536-char cap.
argument-hintShown during autocomplete. Example: [issue-number]
disable-model-invocationSet true to prevent Claude from triggering this skill automatically. Use for side-effect tasks you control.
user-invocableSet false to hide from the / menu. Claude can still invoke it; users cannot.
allowed-toolsTools Claude can use without per-use approval when this skill is active. Space-separated string or YAML list.
modelModel to use when this skill is active.
effortEffort level for this skill. Options: low, medium, high, max.
contextSet fork to run in an isolated subagent. Skill content becomes the task prompt.
agentWhich subagent type to use when context: fork is set. Options: Explore, Plan, general-purpose, or a custom agent name.
hooksHooks scoped to this skill's lifecycle only.
pathsGlob patterns. When set, Claude only auto-loads the skill when working with matching files.
shellShell for inline !`command` blocks. bash (default) or powershell.

Dynamic Context Injection

  • Inline: !`command` — runs before skill content reaches Claude; output replaces placeholder.
  • Multi-line: fenced block opened with ```!
  • Disable for untrusted skills: set disableSkillShellExecution: true in settings.

Compaction Behavior

  • After auto-compaction, the most recently invoked skills are re-attached.
  • Each re-attached skill: up to 5,000 tokens.
  • Combined budget: 25,000 tokens. Oldest skills can be dropped.
  • If a skill stops influencing behavior after compaction, re-invoke it.
