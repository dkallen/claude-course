From Concept to Delivery with Claude Code

Module 7: Skills

Skills turn repeated instructions into invocable slash commands. Instead of re-explaining the same thing across sessions, you write it once as a Markdown file and call it with a single command. This module covers how the skill system works, how to build skills that hold up across different inputs, and where they belong.
