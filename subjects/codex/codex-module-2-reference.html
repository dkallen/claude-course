AGENTS.md Quick Reference

Use It For

  • repo layout
  • build/test/lint commands
  • review expectations
  • constraints and do-not rules
  • verification standards

Do Not Use It For

  • one-off task instructions
  • temporary branch-specific notes
  • long speculative essays

Discovery Idea

Global defaults under ~/.codex, shared repo guidance at root, more specific guidance closer to the current directory.

Best Habit

When Codex makes the same mistake twice, update AGENTS.md so the fix persists.
