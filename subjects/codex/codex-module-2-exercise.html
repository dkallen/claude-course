Module 2: Guided Exercise

Write and test an AGENTS.md in a real repository.

Part 1: Create a Small File

At repo root, write an AGENTS.md that includes:

  • how to run the project
  • how to test it
  • one or two conventions that matter
  • one explicit do-not rule

Part 2: Ask Codex to Summarize the Loaded Guidance

Use a simple prompt asking Codex what instructions it is operating under.

Part 3: Give It a Small Task

Pick something like a small refactor or test addition and see whether it follows the guidance without you repeating it.

Part 4: Tighten the File

If Codex still makes a predictable mistake, update AGENTS.md and rerun a similar task.
