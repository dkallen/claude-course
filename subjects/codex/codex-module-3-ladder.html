Module 3: Planning Ladder

Choose the lightest prompt shape that matches the task.

1. Direct

Use for narrow tasks that are easy to verify.

2. Direct + Constraints

Use when the task is still bounded but there are important do-not rules or acceptance criteria.

3. Plan First

Use when the task is ambiguous, complex, or hard to describe cleanly in one pass.

4. Structured Execution Plan

Use for larger efforts where multiple steps, files, and decisions must stay coordinated.
