Module 4: Guided Exercise

Use Claude Code to refine one real idea before you write any code.

Goal: Produce a written brief with goals, problems, solutions, risks, and scope.

Time estimate: 30–40 minutes

Part 1: Choose a Real Idea

Pick something you actually might build. Avoid toy ideas if possible.

Part 2: Build the Goal Hierarchy

Ask Claude to help you articulate the top-level goal and the contributing goals below it.

Part 3: Identify Problems

List the current obstacles to those goals. Ask Claude which look like root causes and which are likely symptoms.

Part 4: Propose Solutions

Generate several solutions. For each one, run the alignment check: does it solve a real problem or directly advance a real goal?

Part 5: Add Risks and Scope

For the solutions you keep, identify forecasted problems and mitigations. Then define what is in scope and what is out.

Part 6: Capture the Brief

Your final artifact should include:

  • a goal hierarchy
  • named problems
  • aligned solutions
  • risks and mitigations
  • explicit scope
