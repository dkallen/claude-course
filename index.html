Multi-Subject Course Platform

Interactive courses for learning how to work effectively with coding agents.

This platform supports multiple subjects through a shared course model. The current subjects are Claude Code and OpenAI Codex. Use this page as the entry point, then open the course shell for the subject you want to study.

The course shell reads shared subject data from course-data.js. Learner-facing material is standalone HTML and SVG, so the course works directly in the browser on any device.
