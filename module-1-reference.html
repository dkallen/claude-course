Orientation: What Claude Code Actually Is

The Agentic Loop

ReadGathers context — files, code, prompt, CLAUDE.md
PlanForms an approach, weighs options, may ask questions
ActExecutes — writes code, edits files, runs commands
VerifyChecks results — errors trigger a new cycle
Self-correcting within its current approach. If the approach is wrong, you need to redirect.

Context Windows & Compaction

Window size~200K tokens (large, not infinite)
CompactionAutomatic summarization of old content when window fills
SymptomsContradicts earlier decisions, repeats work, re-asks answered questions
FixStart fresh sessions; put critical context in CLAUDE.md (Module 2)
Think: whiteboard with a fixed size. Oldest details get erased first. CLAUDE.md never gets erased.

The Jagged Capability Frontier

Excels At

  • Understanding existing code
  • Boilerplate & standard patterns
  • Clear-intent refactoring
  • Multi-file coordination
  • Documentation generation
  • Debugging from error messages

Struggles With

  • Novel algorithms & complex logic
  • Long-session consistency
  • Knowing what it doesn't know
  • UI/UX judgment & taste
  • Niche frameworks & domains
  • Security edge cases

Trust Calibration

Common pattern? Yes → high trust
Requires judgment? Yes → verify carefully
Easy to verify? Yes → let it try, then check
Cost of error? Low → let it run. High → verify all.
