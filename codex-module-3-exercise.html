Module 3: Guided Exercise

Rewrite one real task so Codex gets the right amount of planning guidance.

Part 1: Pick a Real Task

Choose a task you actually want Codex to help with. It can be small or medium-sized, but it should be real.

Part 2: Classify It

Decide whether the task is direct, direct with constraints, or plan-first. Explain why.

Part 3: Write the Prompt

Include goal, relevant context, constraints, and done-when. If the task is plan-first, ask Codex to gather context and build a plan before implementation.

Part 4: Run It

Observe whether Codex asks clarifying questions, follows the right scope, and stops at a clearly reviewable result.

Part 5: Improve It

If the task drifted, decide whether the problem was missing context, weak constraints, or missing completion criteria.
